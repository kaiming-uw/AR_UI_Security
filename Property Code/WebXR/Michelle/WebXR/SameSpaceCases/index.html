Augmented Reality with the WebXR Device API

This is an experiment using augmented reality features with the WebXR Device API. Upon entering AR, there will be a blue circle on the ground. Tap the screen to create cubes. Upon tapping on a cube, there should be a box that appears on top of it showing its name.
